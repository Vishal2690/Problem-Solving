Question 1 : Find the Smaller Angle

Ans:- 

        var Minimal_Angle = (h, m) => 
        {
          let hrs = (h % 12 + m / 60)*30;
          let min = m * 6;
          let minangle = Math.abs(hrs - min);
          return minangle;
        };

Question 3 : Perfect Number Check."

Ans:- 

        var Perfect_Check = (N) => 
        {
            let add = 0;
            for(let i=1; i< N; i++){
              add += i;
            }
            if(add == N){
              return "YES";
            }
            else{
              return "NO";
            }
                
        };
